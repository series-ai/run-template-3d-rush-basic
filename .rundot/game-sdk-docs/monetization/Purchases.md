# Purchases API (RunBucks)

## Methods
- `getHardCurrencyBalance()` - Check balance
- `spendCurrency(itemId, amount)` - Process spending
- `openStore()` - Open native store
- `getCurrencyIcon()` - Get currency icon
- `getSubscriptions()` - Get subscription tiers
- `purchaseSubscription(tier, interval)` - Purchase subscription
- `isUserSubscribed(tier)` - Check subscription (respects hierarchy)
- `hasUserMadePurchase()` - Check purchase history
